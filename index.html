Projects

Transacto

HTML, CSS, JavaScript, Flask, MySQL

The project aims to bolster payment security in response to rising credit card fraud. It involves verifying transactions by utilising MAC addresses, which are unique hardware identifiers. The authenticity integrity of each transaction is ensured by confirming the payment request origin as trusted device. The application is built with the Flask micro-framework, providing a lightweight yet modular architecture that facilitates future scalability and seamless integration with the frontend.

Mededge

React, Vite, CSS, Flask, Pyramid, SQLite, SQLAlchemy

The project is a healthcare website to automate medical systems using edge computing for reduced latency and integration of IOT devices. – The cloud server behaves as the metadata server and the unversities and hospitals are the edge nodes which interact directly to the users. – Flask and Pyramid were used to develop RESTful APIs for data interaction, while Tornado was used for handling asynchronous tasks and real-time communication.

Real-Time Product Recommender

Python, Flask, YOLO, ResNet-50, OpenCV, HTML, CSS, JS

The project focuses on real-time object detection from a paused frame of a video player and delivering similar product recommendations, similar to Amazon Prime Video’s ”X-Ray feature.” ∗ The backend was implemented using the Flask micro-framework, integrated seamlessly with the frontend. We used Base64 encoding to transmit image data across endpoints efficiently. ∗ The ML model uses the YOLO model for object detection ResNet-50 model for feature extraction. Product recommendations are generated from a database by calculating similarity using the Cosine-Similarity metric.
